Владислав (22 года, инженер-программист, магистратура) и Александр совместно разрабатывают образовательную и инженерную экосистему Equilibrium — репозиторий с проектами по процедурному моделированию, CNC, 3D-печати и автоматизации.
Структура репозитория:
equilibrium/
├── Houdini-Learning/          # Курс по SideFX Houdini (в корне!)
│   ├── Theory/                # Теория: интерфейс, ноды, динамика, VEX
│   ├── Practice/              # Практика: 7 модулей (01–07)
│   ├── Progress/              # vlad-progress.md, alexander-progress.md
│   ├── Resources/             # Glossary-RU.md
│   └── Roadmap.canvas         # Визуальная карта обучения
├── Projects/
│   ├── 01-CNC-Laser/          # Лазерный станок (планируется к покупке)
│   ├── 02-3D-Printer/         # 3D-печать (планируется к покупке)
│   ├── 03-Data-Analytics/     # Скрипты обработки данных (Python)
│   └── 04-AI-Assistant/       # Эксперименты с ИИ
├── Docs/Tasks/                # Центральный трекер задач (Tasks.md)
├── .obsidian/                 # Настройки Obsidian (полностью синхронизируются через Git)
└── ...                        # остальные служебные папки
Технологии и инструменты:
 1. Obsidian с плагинами: Tasks, Note Toolbar, Templater, Dataview, Advanced Canvas, Admonition, Kanban, Git, Editing Toolbar, Emoji Toolbar, Iconize
 2. Git — полная синхронизация конфигурации Obsidian (включая community-plugins.json и настройки плагинов)
 3. SideFX Houdini 20.5–21.0 — обучение и визуализация
 4. Windows (RTX 3060, 64 ГБ RAM, AMD R7 5700G)
Реализовано:
✅ Курс Houdini: 7 модулей (от интерфейса до интеграции с CNC/3D-печатью)
✅ Глоссарий терминов (Glossary-RU.md) — с разделами, рабочими ссылками, кнопкой «назад»
✅ Централизованный трекер задач (Docs/Tasks/Tasks.md) — на базе Obsidian Tasks с глобальным фильтром #task
✅ Система прогресса — файлы vlad-progress.md и alexander-progress.md
✅ Obsidian-инфраструктура: шаблоны, Note Toolbar, синхронизация настроек через Git
Особенности:
Все задачи помечаются тегом #task
Используется команда «Tasks: Toggle Done» для завершения задач
Физическое оборудование (CNC, 3D-принтер) ещё не приобретено — работа ведётся на этапе симуляции и планирования
Проект Equilibrium — это сам репозиторий, а не отдельная папка
Цель: создание портфолио, автоматизация инженерных задач, совместное обучение, подготовка к работе с реальным оборудованием.
