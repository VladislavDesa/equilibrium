# Глоссарий терминов Houdini (RU ↔ EN)
| **Общие понятия** 
| Русский | Английский | Пояснение |
|--------|------------|----------|
| Нода | Node | Базовый строительный блок сети в Houdini |
| Сеть | Network | Граф соединённых нод (SOP, DOP, VOP и др.) |
| Геометрия | Geometry | Данные о форме объекта: точки, примитивы, атрибуты |
| Контекст | Context | Тип сети: SOP, DOP, VOP, CHOP, ROP |
| Viewport | Viewport | Окно 3D-просмотра сцены |
| Network Editor | Network Editor | Область создания и редактирования сетей |
| Parameter Editor | Parameter Editor | Панель настройки параметров ноды |
| Guide Geometry | Guide Geometry | Вспомогательная визуализация (нормали, центры масс  и т.д.) |
| HIP / HIPNC | .hip / .hipnc | Формат проекта Houdini (HIPNC — без бинарных данных) |

| **Геометрия и структура** |
| Точка | Point | Позиция в 3D-пространстве (`@P`) |
| Вершина | Vertex | Связь между точкой и примитивом |
| Примитив | Primitive | Элемент геометрии: полигон, меш, объём, кривая |
| Деталь | Detail | Глобальные данные всей геометрии |
| Атрибут | Attribute | Дополнительные данные у точки/примитива (`@Cd`, `@N`, `@v`) |
| Normal | Normal | Нормаль — направление поверхности |
| UV | UV | Координаты текстурирования |
| PolyReduce | PolyReduce | Уменьшение количества полигонов |
| Subdivide | Subdivide | Увеличение детализации меша |
| Boolean | Boolean | Операции объединения/вычитания геометрии |
| Scatter | Scatter | Распределение точек по поверхности |
| Copy to Points | Copy to Points | Копирование геометрии на точки |
| Merge | Merge | Объединение нескольких геометрий |
| Null | Null | Служебная нода-маркер, без геометрии |
| Switch | Switch | Переключение между входами |
| Sticky Note | Sticky Note | Комментарий в сети |

| **VEX и программирование** |
| VEX | Vector Expression Language | Язык программирования в Houdini |
| Wrangle | Wrangle | Нода для выполнения кода на VEX |
| @P | @P | Атрибут позиции точки (vector) |
| @Cd | @Cd | Атрибут цвета (vector) |
| @N | @N | Атрибут нормали (vector) |
| @ptnum | @ptnum | Номер текущей точки (int) |
| @primnum | @primnum | Номер текущего примитива (int) |
| @Time | @Time | Глобальное время симуляции (float) |
| rand() | rand() | Генератор случайных чисел |
| noise() | noise() | Шумовая функция (Perlin) |
| fit() | fit() | Преобразование диапазона значений |
| removepoint() | removepoint() | Удаление точки из геометрии |
| Run Over | Run Over | Контекст выполнения VEX: point, primitive, detail и др. |

| **Динамика и симуляции** |
| DOP | Dynamics Operator | Контекст для физических симуляций |
| SOP | Surface Operator | Контекст для работы с геометрией |
| POP | Particle Object Protocol | Подсистема для работы с частицами |
| RBD | Rigid Body Dynamics | Симуляция твёрдых тел |
| Fracture | Fracture | Разрушение объекта на осколки |
| Solver | Solver | Нода, выполняющая расчёт симуляции во времени |
| Gravity | Gravity | Гравитация в симуляции |
| Static Object | Static Object | Неподвижный объект-коллайдер |
| Collision | Collision | Столкновение объектов |
| Cache | Cache | Сохранение симуляции на диск для ускорения |
| Trail | Trail | Создание следа из движущихся точек |
| Carve | Carve | Обрезка геометрии по параметру U |
| VDB | VDB | Формат объёмных данных (дым, жидкость, облака) |

| **Интеграция и экспорт** |
| G-code | G-code | Язык управления станками (CNC, 3D-печать) |
| STL | STL | Формат файла для 3D-моделей |
| OBJ | OBJ | Универсальный формат 3D-геометрии |
| Overhang | Overhang | Участок модели, выступающий без опоры (в 3D-печати) |
| Support | Support | Поддержки для overhang-зон при печати |
| Layer Height | Layer Height | Высота слоя при 3D-печати |
| Feed Rate | Feed Rate | Скорость подачи инструмента (CNC) |
| Kerf | Kerf | Ширина пропила лазера/фрезы |
| Toolpath | Toolpath | Траектория движения инструмента |
| Preview | Preview | Предварительный просмотр перед запуском |

| **Система и производительность** |
| Frame | Frame | Кадр анимации |
| Time | Time | Глобальное время симуляции |
| FPS | FPS | Кадров в секунду (в симуляции) |
| Cook | Cook | Процесс пересчёта сети |
| Dirty Flag | Dirty Flag | Метка, что нода требует пересчёта |
| Lazy Evaluation | Lazy Evaluation | Отложенный расчёт (только при необходимости) |
